Projects Overview Fire Simulator

Forest Fire Simulator

Project Information:

  • Built in 2026 for the Kelowna Fire Department as my Computer Science capstone project.
  • Built in Unity Game Engine with Model View Controller architecture, using an Agile, CI/CD approach.
  • Uses Unity Testing Framework for automated Unit testing.
  • Uses cellular automata with a Moore's neighbourhood.
  • Probability accounts for slope, fuel load, wind speed and direction, humidity, and temperature.
